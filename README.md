📌 项目简介（Introduction）
English: a powerful and all-in-one prompt management solution for ComfyUI. It provides a top-bar quick-access button, a complete visual prompt browser, and highly customizable UI to help you manage, preview, and share prompts efficiently.
中文: 为 ComfyUI 打造的提示词管理方案。集成顶栏快捷访问按钮、完整的可视化提示浏览器与高度自定义 UI，助你高效管理、预览与共享提示词。

✨ 核心功能（Core Features）

🌟 界面与交互
- **顶栏快捷入口**：在 ComfyUI 顶部菜单栏添加专属 Prompt Manager 快速访问按钮，一键打开管理界面。
- **完整 Prompt 浏览器**：提供全局、系统化的提示词浏览与管理界面，直观对比不同模型下提示词的效果，提高创作效率。
- **高度自定义 UI**：支持自定义 UI 配色主题、设置UI背景、护眼遮罩与卡片透明度调节，适配不同审美与工作环境。

📦 数据管理与共享
- **批量操作**：支持批量导入图片、导出 JSON 数据、多选卡片进行统一删除、移动、收藏、组合等操作。
- **智能 Tag 导入**：自动识别并解析 URL 编码内容，一键快捷导入 Tag。
- **多层级分类管理**：支持文件夹式的多级分类与子分类，轻松管理海量提示词。

🛒 创新组合功能
- **购物车组合串**：可以将不同分类、不同模型的提示词卡片加入购物车，自由组合成提示词组合串，一键应用到工作流。
- **收藏夹管理**：支持收藏夹管理、重命名、快速切换，打造个人专属提示词预览库。

🚀 快速安装（Quick Start）

**方法一：ComfyUI-Manager（推荐）**
1. 打开 ComfyUI，进入 ComfyUI-Manager 插件管理界面。
2. 搜索 Prompt Manager，点击安装。
3. 重启 ComfyUI，在顶部菜单栏即可看到 Prompt Manager 按钮。

**方法二：手动安装**
1. 下载本仓库源码。
2. 将整个文件夹解压到 ComfyUI/custom_nodes/ 目录下。
3. 重启 ComfyUI，在顶部菜单栏即可使用。

📖 使用指南（Usage）

1. **打开界面**：点击 ComfyUI 顶栏的粉色 P 字样按钮，打开主界面（shift点击可以在ComfyUI界面中打开小窗口）。
2. **基础管理**：在左侧分类栏新建文件夹/子分类，点击「新建卡片」输入提示词、添加标签、上传预览图。
3. **多模型对比**：在卡片详情或列表页，顶部切换模型列表，直观看出不同模型对同一提示词的输出差异。
4. **批量操作**：勾选多个卡片，底部出现批量操作栏，可移动、删除或组合；点击「导入」选择批量图片或粘贴含 URL 编码的文本。
5. **购物车功能**：选中卡片点击「加入购物车」，在右侧购物车标签页查看，可组合成"组合串"一键应用到工作流。

⚙️ 配置选项（Settings）

- 自定义UI主题色（日间、夜间模式切换）
- 自定义UI背景
- 是否启用背景预览图、护眼遮罩
- 卡片默认透明度
- 配置图片压缩参数
