# 📌 Prompt Manager

English: A powerful and all-in-one prompt management solution for ComfyUI. It provides a top-bar quick-access button, a complete visual prompt browser, and highly customizable UI to help you manage, preview, and share prompts efficiently.

中文: 为 ComfyUI 打造的提示词管理器。集成顶栏快捷访问按钮、完整的可视化提示浏览器与高度自定义 UI，助你高效管理、预览与共享提示词。

---

## ✨ 核心功能（Core Features）

### 🌟 界面与交互
- **顶栏快捷入口**：在 ComfyUI 顶部菜单栏添加专属 Prompt Manager 快速访问按钮，一键打开管理界面。
- **完整 Prompt 浏览器**：提供全局、系统化的提示词浏览与管理界面，直观对比不同模型下提示词的效果，提高创作效率。
- **高度自定义 UI**：支持自定义 UI 配色主题、设置 UI 背景、护眼遮罩与卡片透明度调节，适配不同审美与工作环境。
- **夜间模式**：一键切换日间/夜间模式，保护视力。
- **多标签页管理**：支持同时打开多个分类标签页，快速切换不同分类。

### 📦 数据管理与共享
- **批量操作**：支持批量导入图片、导出 JSON 数据、多选卡片进行统一删除、移动、收藏、组合等操作。
- **智能 Tag 导入**：自动识别并解析 URL 编码内容，一键快捷导入 Tag。
- **多层级分类管理**：支持文件夹式的多级分类与子分类，轻松管理海量提示词。
- **管理模式**：支持管理分类，自由创建、编辑、删除分类结构。
- **数据本地化存储**：所有数据存储在 `ComfyUI\custom_nodes\ComfyUI-Prompt-Manager\data` 目录下，便于管理和备份。

### 🛒 创新组合功能
- **购物车组合串**：可以将不同分类、不同模型的提示词卡片加入购物车，自由组合成提示词组合串，一键应用到工作流。
- **收藏夹管理**：支持收藏夹管理、重命名、快速切换，打造个人专属提示词预览库。
- **综合分组**：灵活的分组管理功能，便于整理和归类提示词。

### 🔍 搜索与筛选
- **多关键字模糊搜索**：支持多关键字同时搜索，快速定位目标提示词。
- **范围筛选**：可按当前小分类、当前大分类或全部进行筛选。
- **备注搜索**：支持按备注内容搜索。
- **标签筛选**：通过标签快速筛选相关提示词。
- **状态筛选**：按状态筛选提示词。
- **排序功能**：支持按名称升序等多种排序方式。
- **缩放调节**：卡片大小自由缩放，适应不同浏览习惯。

### 📦 ComfyUI 节点集成
- **Prompt 浏览器节点**：可点击下方的"打开 Prompt 浏览器"按钮打开管理器小窗口，快捷把提示词导入到此节点文本框中。
- **Prompt 一键导入节点**：可以直接把生成的预览图附上提示词快捷导入到对应管理器目录下。

## 🚀 快速安装（Quick Start）

### 方法一：ComfyUI-Manager（推荐）
1. 打开 ComfyUI，进入 ComfyUI-Manager 插件管理界面。
2. 搜索 `Prompt Manager`，点击安装。
3. 重启 ComfyUI，在顶部菜单栏即可看到 Prompt Manager 按钮。

### 方法二：手动安装
1. 下载本仓库源码。
2. 将整个文件夹解压到 `ComfyUI/custom_nodes/` 目录下。
3. 重启 ComfyUI，在顶部菜单栏即可使用。
4. 确保安装后路径为`ComfyUI/custom_nodes/ComfyUI-Prompt-Manager`，文件名保持一致。

## 📖 使用指南（Usage）

### 基础使用流程

1. **顶部标签页**：可按照模型种类（或个人喜好）分类。

2. **创建新卡片**：在左侧卡片管理中点击「导入/创建卡片」，可通过以下两种方式创建：
   - **图片上传**：若无卡片则会自动读取文件名作为卡片名称(prompt)。可直接上传压缩包内按模型分类好的图片。
     - 如果 Prompt 中含有电脑文件名无法保存的特殊符号（如 `\ / : * ? " < > |`），请先对其进行 URL 编码（如转为 `%40` 等）再作为文件名上传，系统将自动帮您解码显示。
   - **文本上传**：单个/批量创建无图片的新卡片（输入框直接支持保存包含上述所有特殊符号的 Prompt）。

3. **复制 Prompt**：点击卡片名称(prompt)即可复制，若想要多个复制则加入购物车，在右下角购物车中点击"导出为 prompt"即可。

4. **关于多图**：如果需上传多张图，卡片生成后点击空占位图或最后方的"+"可多选批量上传追加新图！

### 收藏功能
- 点击卡片内的 ⭐ 即可将常用的卡片归类到专属的分组收藏夹中。
- 您可在侧边栏或右下角快速打开收藏夹，方便日后针对不同画风快速调取卡片。

### 组合功能
- 想混合多个画师的画风？进入「组合管理」，您可以创建诸如 `(画师A:1.2), (风格B:0.8)` 这样的组合预设。
- 为这个组合单独上传一张总预览图，方便未来一键调用复杂的 Prompt 串。

### 购物车功能
- 用于暂时存放您挑选好的卡片。
- 可一键合并导出所有的 Prompt（并支持单独微调每个 Prompt 的权重）。

### 卡片管理
- **导入/创建卡片**：
  - **图片批量上传**：选取多张图片，系统将以去后缀的文件名自动创建并上传（若名称包含不支持的特殊符号，请将文件名进行 URL 编码）。
  - **单个文本创建**：手动创建单个提示词卡片。
  - **TXT 批量导入**：通过 TXT 文件批量导入提示词。
- **批量选择与操作**：勾选多个卡片，可进行批量移动、删除、收藏、组合等操作。
- **清理所有图片**：一键清理所有卡片中的预览图片。

### ComfyUI 节点使用

#### Prompt 浏览器节点
- 在 ComfyUI 工作流中添加「Prompt 浏览器」节点。
- 点击节点下方的"打开 Prompt 浏览器"按钮，将打开管理器小窗口。
- 在管理器中选择需要的提示词，点击后会自动导入到节点的文本框中。
- 节点输出的 `prompt_string` 可直接连接到其他需要提示词的节点。

#### Prompt 一键导入节点
- 在 ComfyUI 工作流中添加「Prompt 一键导入」节点。
- 连接生成的图像到 `images` 输入端口。
- 输入提示词到 `prompt_text` 输入端口。
- 选择目标分类器列表中的分类目录。
- 运行节点后，系统会自动将图像和提示词导入到对应的管理器目录中。

## 📊 数据管理（Data Management）

### 导入数据
- **导入数据 (JSON)**：从 JSON 文件导入提示词数据。
- **智能映射导入**：别人导出的分类，您在导入时可强行映射给自己的任意标签页，无需刻板遵守原始路径！

### 导出数据
- **导出数据中心 (JSON)**：
  - 仅当前小分类数据（单模式）
  - 当前大分类（涵盖旗下所有小分类）
  - 当前标签页全库（强烈推荐用于全备份）
  - 可选择是否包含图片导出
  - ⚠️ **警告**：如果您的图库极大（超过几百 MB），包含图片的全量导出可能会触发浏览器单字符串内存限制导致失败。建议超大图库分大类导出。
- **导出名称列表 (TXT)**：导出提示词名称列表为 TXT 文件。

### 备份管理
- 内置备份管理功能，保障数据安全。
- 所有数据存储在 `ComfyUI\custom_nodes\ComfyUI-Prompt-Manager\data` 目录下。

## ⚙️ 设置与帮助（Settings & Help）

### UI 主题与背景
- **UI 主题色**：自定义界面主题颜色，支持恢复默认。
- **界面边角风格**：圆角风格（默认）等多种选择。
- **夜间模式**：一键切换日间/夜间模式。
- **自定义图片背景**：选择本地图片作为背景（优先级最高）。
- **卡片与侧边栏透明度**：0-100% 自由调节（默认 95%）。
- **暗化遮罩程度**：防刺眼遮罩，0-100% 自由调节（默认 50%）。
- **背景毛玻璃模糊**：0-50px 自由调节（默认 10px）。
- **右下角悬浮小红点**：显示/隐藏组合/收藏/购物车的小红点提示。

### 图片压缩设置
- **压缩质量**：0-100% 自由调节（默认 85%），平衡图片质量与存储空间。
- **最大宽度**：限制图片最大宽度（默认 900px），超出将自动等比例缩放，优化加载性能。
