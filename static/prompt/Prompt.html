📖 Prompt浏览器 使用手册
1. 基础使用流程
①. 顶部为标签页，可按照模型种类（或个人喜好）分类。
②. 在左侧卡片管理中点击创建新卡片，可通过图片上传/文本上传两种模式创建新卡片（可直接上传压缩包内按模型分类好的图片）。
    图片上传：若无卡片则会自动读取文件名作为卡片名称(prompt)。如果Prompt中含有电脑文件名无法保存的特殊符号（如 \ / : * ? " < > |），请先对其进行 URL 编码（如转为%40等）再作为文件名上传，系统将自动帮您解码显示。
    文本上传：单个/批量创建无图片的新卡片（输入框直接支持保存包含上述所有特殊符号的 Prompt）。
③. 点击卡片名称(prompt)即可复制，若想要多个复制则加入购物车，在右下角购物车中点击“导出为prompt”即可。

✨ 关于多图：如果需上传多张图，卡片生成后点击空占位图或最后方的“+”可多选批量上传追加新图！
2. 收藏功能介绍
点击卡片内的 ⭐ 即可将常用的卡片归类到专属的分组收藏夹中。您可在侧边栏或右下角快速打开收藏夹，方便日后针对不同画风快速调取卡片。
3. 组合功能介绍
想混合多个画师的画风？进入“组合管理”，您可以创建诸如 (画师A:1.2), (风格B:0.8) 这样的组合预设，并为这个组合单独上传一张总预览图，方便未来一键调用复杂的 Prompt 串。
4. 购物车功能介绍
用于暂时存放您挑选好的卡片。可一键合并导出所有的 Prompt（并支持单独微调每个 Prompt 的权重）。
5. 导出与导入引擎
①. 数据导出：支持自由选择导出范围（单分类/大类/全库）以及是否包含图片。
②. 智能映射导入：别人导出的分类，您在导入时可强行映射给自己的任意标签页，无需刻板遵守原始路径！
